# Nebula Flow V7

Nebula Flow V7 is a premium Flutter Web portfolio for Moe Kyaw Aung.

## Features
- Glassmorphism layout
- Responsive hero section
- About, projects, case studies, experience, certificates, testimonials, and contact
- Modular widget-based structure
- Flutter Web friendly

## Tech Stack
- Flutter
- Dart
- Material 3
- url_launcher

## Folder Structure
- lib/screens
- lib/widgets
- lib/models
- lib/data
- lib/theme
- lib/utils

## Getting Started
1. Install Flutter.
2. Run `flutter pub get`.
3. Run `flutter run -d chrome`.

## Build
```bash
flutter build web
